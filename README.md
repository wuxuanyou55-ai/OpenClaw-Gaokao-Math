## ✨ 核心亮点
- **游戏化学习**：30关阶梯式闯关，星级评价、章节奖励，抓住学生过关心理
- **视频式讲解**：借鉴 VideoTutor 风格，每关先讲原理本质，再练题，拒绝套路
- **第一性原理**：从定义→推导→题型→真题，一条线打通，理解才会做题
- **适配 OpenClaw**：一键安装即可使用，无需额外配置

## 📦 安装方法（一键复制）
```bash
# 1. 克隆仓库
git clone https://github.com/wuxuanyou55-ai/OpenClaw-Gaokao-Math.git
cd OpenClaw-Gaokao-Math

# 2. 安装到 OpenClaw 技能目录
mkdir -p ~/.openclaw/skills/gmath
cp openclaw-skill/SKILL.md ~/.openclaw/skills/gmath/

# 3. 刷新技能
openclaw skills refresh
